This page is to exercise an effect commonly executed using JS methods.
I have done it here using CSS to streamline it and cut down on rendering time.


Note that while the page scrolls, the background image remains static





Text can be separated into handy blocks to make clean lines and a smooth transition between opposing bg images




Multiple divs can each have the same bg or their own image to incorporate different images throughout the page.

This is effective for telling a story about a business through images which accompany text.

The text is large to show another possible use for this.

If we directly follow one static bg image with another, it creates another interesting effect.

The following image will appear to slide over the top of the existing image.

This effect does depend a bit on the images being scrolled between.

The entirety of this effect is achieved through CSS so rendering time is kept as low as possible.


Another advantage to taking this approach is that it can be used on cookie cutter sort of sites where a user is assigned a default page and is able to edit individual html blocks and CSS but not enter their own JS.
